# PRD — AI Budget Coach

## 1. Product Overview
AI Budget Coach is a simple web application that helps users understand and improve their financial habits using artificial intelligence. Users enter their income and expenses, and the system analyzes the data to provide insights, recommendations, and a financial health score.

## 2. Problem Statement
Many students and young adults struggle to manage their finances effectively. They often do not know where their money goes and cannot identify unnecessary spending. Existing solutions focus mainly on tracking expenses but lack personalized insights and guidance.

## 3. Target Users
- University students  
- Young adults  
- New graduates  
- Users with basic financial knowledge  

## 4. Value Proposition
AI Budget Coach provides:
- Fast and simple financial analysis  
- Personalized AI-driven recommendations  
- Clear and easy-to-understand insights  
- A beginner-friendly budgeting experience  

## 5. Core Features
1. User inputs income and expenses  
2. AI analyzes financial data  
3. Expenses are categorized  
4. Unnecessary spending is identified  
5. Savings suggestions are generated  
6. Financial health score is calculated (0–100)  
7. A short budget improvement plan is provided  

## 6. User Input
Users can enter:
- Monthly income  
- Expense categories (rent, food, coffee, etc.)  
- Optional notes about spending  

## 7. System Output
The system returns:
- Spending summary  
- Category breakdown  
- Financial health score  
- Unnecessary spending alerts  
- Savings recommendations  
- Budget improvement plan  

## 8. User Interface

### Screen 1 — Input Screen
- App title  
- Short description  
- Input fields for income and expenses  
- Analyze button  

### Screen 2 — Results Screen
- Spending summary  
- Category breakdown  
- Financial score  
- AI recommendations  

## 9. User Flow
1. User opens the application  
2. User enters income and expenses  
3. User clicks the analyze button  
4. AI processes the data  
5. Results are displayed  

## 10. Technical Scope
- No login required  
- No database required (prototype level)  
- Single page application  
- AI integration via Gemini API  

## 11. Constraints
- Simple design  
- Fast performance  
- Mobile-friendly  
- Beginner-level implementation  

## 12. Success Criteria
- Users can input data easily  
- AI generates meaningful and useful insights  
- Output is clear and understandable  
- The application works end-to-end (input → AI → output)
