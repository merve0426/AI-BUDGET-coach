# Tech Stack — AI Budget Coach

## Frontend
- HTML
- CSS
- JavaScript

## AI Integration
- Gemini API (Google AI Studio)

## Development Tools
- Cursor / VS Code

## Code Hosting
- GitHub

## Deployment
- Netlify / Lovable

## Why This Stack?

This tech stack is chosen because it is:
- Beginner-friendly
- Fast to build and deploy
- Suitable for a working prototype
- Easy to integrate with AI tools

## How It Works

1. The user inputs financial data (income and expenses)
2. The frontend sends the data to the backend logic
3. The app connects to Gemini API
4. AI processes the data
5. Results are returned and displayed to the user

## Setup Summary

1. Create project files (HTML, CSS, JS)
2. Build user interface
3. Connect Gemini API
4. Test AI responses
5. Push to GitHub
6. Deploy using Netlify or Lovable
