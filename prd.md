# PRD — AI Budget Coach

## Product Overview
AI Budget Coach is a web app that helps users analyze their expenses and improve their financial habits using AI.

## Problem
Users struggle to understand where their money goes and how to manage their budget effectively.

## Target Users
- Students
- Young adults
- Beginners in finance

## Features
- Expense input
- AI analysis
- Category breakdown
- Financial score
- Savings suggestions
- Budget plan

## Input
User enters:
- income
- expenses

## Output
AI returns:
- analysis
- categories
- financial score
- suggestions

## Screens
1. Input screen
2. Result screen

## Goal
Provide fast and simple financial insights using AI
