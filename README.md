# AI Budget Coach

## Problem
Many students and young adults struggle to manage their finances effectively. They often cannot clearly understand where their money goes, which leads to unnecessary spending and poor budgeting habits.

## Solution
AI Budget Coach is a web application that uses artificial intelligence to analyze user spending. It categorizes expenses, detects unnecessary spending, and provides personalized savings recommendations along with a financial health score.

## Features
- Expense and income input
- AI-powered spending analysis
- Category-based breakdown
- Financial health score (0–100)
- Personalized savings suggestions
- Budget improvement plan

## User Flow
1. User enters income and expenses  
2. User clicks the analyze button  
3. AI processes the data  
4. Results are displayed with insights and recommendations  

## Technologies
- HTML, CSS, JavaScript  
- Gemini API  
- GitHub  
- Netlify / Lovable  

## How to Run
1. Open the project  
2. Enter your income and expenses  
3. Click analyze  
4. View AI-generated results
