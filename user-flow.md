# User Flow — AI Budget Coach

## Step 1 — Open Application
The user opens the web application and sees the home screen.

## Step 2 — View Input Section
The user sees input fields for income and expenses along with a short description of the app.

## Step 3 — Enter Financial Data
The user enters:
- income
- expenses (rent, food, coffee, etc.)

## Step 4 — Click Analyze Button
The user clicks the "Analyze" button to process the data.

## Step 5 — AI Processing
The system sends the input data to the AI (Gemini API), which analyzes the financial information.

## Step 6 — Display Results
The system shows:
- spending summary  
- categorized expenses  
- financial health score  
- unnecessary spending alerts  
- savings recommendations  
- budget improvement plan  

## Step 7 — User Review
The user reviews the results and understands their financial situation better.
